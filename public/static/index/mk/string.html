字符串中常用的方法

在JS中用单双引号包裹起来的都是字符串，单双引号中的空格符号等都是一个字符
console.dir(String.prototype)查看全部方法

//以数字为索引，从0开始

var str = '1+2=3'

str[0] -> '1'

str[100] -> 'undefined'

charAt && charCodeAt

str.charAt(0)：返回指定索引对应的字符，和str[0]的区别在于，当指定索引不存在的时候，str[0]获取的是undefined，而charAt(0)获取的是空字符串
str.charCodeAt(0)：在charAt的基础上，把获取到的字符变为unicode编码值（对应ASCII码表）

substr && substring && slice

实现字符串截取的三个方法
str.substr(n,m)：从索引n开始，截取m个字符
str.substring(n,m)：从索引n开始，截取到索引m（不包含m），把找到的部分截取
str.slice(n,m)：和substring语法一样，区别在于slice可以以负数做索引

var str = '12345+67890';

str.substr(1,5) ->"2345+"

str.substring(1,5) ->'2345'

str.slice(1,5) ->'2345'

str.slice(-3,-1) ->'89' ->str.slice(str.length-3,str.length-1)

当索引是负数的时候，浏览器在处理的时候，是用字符串的总长度加上负数索引，然后按照正数处理操作
细节：
1.如果只传递了n，（str.substr(n)/str.substring(n)）,相当于从索引n开始一直截取到字符串的末尾
2.如果传递的索引超出最大限制，也是把能截取到的部分截取掉而已（有多少截多少）
3.如果一个参数都不传递，相当于将整个截取（字符串的克隆）

toUpperCase && toLowerCase

str.toUpperCase()：把字母全部大写
str.toLowerCase()：把字母全部小写

indeOf && lastIndexOf

str.indexOf(”)：获取当前字符在字符串中第一次出现的位置的索引
lastIndexOf(”)：获取的是最后一次出现的位置的索引
如果当前字符在字符串中没有出现过，结果为-1

if(str.indexOf('?') === -1){

    //当前字符串中没有出现'？'

}

split

str.split(”)：按照某一个字符把字符串拆分成数组中的某一项，支持正则，和数组中的join方法是对应的
如果字符不存在于字符串中，那将整个字符串当做一项进行拆分

replace

str.replace(‘需要替换的’,’替换成为’)：实现字符的替换
在不使用正则的时候，执行一次replace只能替换一次

trim && trimLeft && trimRight

str.trim()：去除字符串首尾的空格
str.trimLeft()：去除字符串开始的空格
str.trimRight()：去除字符串结尾的空格
