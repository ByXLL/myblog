条件？条件成立执行：条件不成立执行
简单if、else判断的另外一种写法

等同于：

if(条件){
}else{
}

var a = 3;

if(a>1 && a<=5){

    a++;

}else{

    a--;

}

//等同于

num>1 && a<=5 ? a++ : a--;



if(a>1 && a<=5){

    a++;

}

num>1 && a<=5 ? a++ : ;     //语法报错



//如果只有if没有else，使用null、undefined、void 0，来占位完善语法

num>1 && a<=5 ? a++ : null;

多事件处理

当我们在判断完成需要执行多件事情的时候，在三元表达式中，可以用小括号括起来，用逗号连接。
但是不能出现，break、continue、return，这些关键字，否则语法报错。

if(a>1 && a<=5){

    a++;

    console.log(a);

}

num>1 && a<=5 ? (a++,console.log(a)) : null;

嵌套

var a = 10；

if(a>0){

    if(a>20){

        a++;

    }else{

        a+=2;

    }

}else{

    if(a>-5){

        a=0;

        alert(a);

    }

}

//转换成三元表达式

a>0?(a>20?a++:a+=2):(a>-5?(a=0,alert(a)):null);
