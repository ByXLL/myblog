switch case

switch case应用于if、else中一个变量在不同值情况下的不同操作

基本语法

var num = 10;

switch(num){

    case 1:

        ......;

        break;

    case 10:

        ......;

        break;

    default:

        ......;

}

switch case 中比较使用的是 === 全等。
switch 后面小括号中存放的是一个值（一般我们都写变量，把变量存储的值拿来用，有时候也可能是一个计算）
case 后面放的都是值，目的是验证switch()中的值和哪一种case后面的值相等，相等则执行后面的操作。
break 正常使用，每一个case执行完后，最后面要加break跳出判断，最后一项case可以不加。
break 特殊使用，如第一项不加break，不管后面的条件是否成立，都会继续往下执行。
default 当switch（）中的值和每一个case情况对应的值都不相等，执行最后的default，相当于else的作用，可写可不写。

案例

//num%2，让num存储的值除以2取余数（0或者1），

var num = 5;

switch(num%2){  //->先把取余操作进行运算，拿结果与case比较

    case 1:

        num++;

        break;

    case 2-1:   //->case后面也应该是值，此处先把2-1运算，把计算后的值与switch值对比

        num--;

        //break;  ->最后一项可以不加break，也能跳出判断

}

巧妙案例 不加break

当符合某几项值都去执行同一件事情

var num = 4;

switch(num%2){

    case 0:

    case 1:    //->当num%2为0和1时都执行num--

        num--;

        break;

    default:

        num = 0;

}
